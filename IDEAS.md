# Ideas & Feature Requests

All ideas, feature requests, and future enhancements for Synapse are managed as **GitHub Issues**.

## 📋 How to Submit an Idea

1. Go to [Issues](https://github.com/FTMahringer/Synapse/issues/new/choose)
2. Select the **✨ Feature Request** template
3. Fill in:
   - **Problem / Motivation** — What problem does this solve?
   - **Proposed Solution** — How should it work?
   - **Alternatives** — Other approaches considered?
   - **Priority** — Low / Medium / High
4. Submit the issue

## 🏷️ Labels

- `enhancement` — New features
- `future idea` — Ideas not yet planned for implementation
- `documentation` — Documentation-related requests

## ❌ What Not to Do

Do **not** add idea files to the repository. Use the issue tracker instead — it keeps things searchable, discussable, and trackable.
